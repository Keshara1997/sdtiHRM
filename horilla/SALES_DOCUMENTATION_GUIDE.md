## **ðŸ“‹ Common Questions & Answers**

### **Q: Is it really free?**
**A**: Yes, Horilla is open-source with LGPL license. No licensing fees. You only pay for implementation, hosting, and optional support services.

### **Q: Can we customize it?**
**A**: Absolutely! Full source code access allows complete customization. You can modify any feature to fit your needs.

### **Q: What about support?**
**A**: Community support is free. Professional support, training, and customization services are available as optional paid services.

### **Q: How long does implementation take?**
**A**: Typically 8-12 weeks depending on complexity, data migration, and customization requirements.

### **Q: Can it integrate with our existing systems?**
**A**: Yes, Horilla provides RESTful API for integrations. Common integrations include biometric devices, email systems, and other business applications.

### **Q: Is it secure?**
**A**: Yes, enterprise-grade security with role-based access control, audit logging, encryption, and compliance features.

### **Q: How many employees can it handle?**
**A**: Horilla is scalable and can handle organizations from small (10+) to large enterprises (10,000+ employees).

### **Q: Can we host it ourselves?**
**A**: Yes, on-premise deployment is fully supported. You can also deploy on cloud platforms (AWS, Azure, GCP) or use Docker.
